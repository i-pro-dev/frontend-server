# frontend-server
Репозиторий с функциональностью вебсервиса, предоставляющего удобную площадку для поиска будущих сотрудников

### Установка

Для установки требуется наличие Python с версией 3.8 и более. 

1. 
   ```sh
   git clone https://github.com/i-pro-dev/frontend-server.git
   ```
2.
   ```sh
   cd frontend-server
   ```
3. 
   ```sh
   python -m pip install -r requirements.txt
   ```
4.
   ```sh
   python app/app.py
   ```
Также проект полностью готов для портирования на Heroku.
